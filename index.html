Mean-Variance Optimization Research

Modern portfolio optimization with machine learning enhancements

Status: Active Research

O Overview

This research project extends the classical Markowitz mean-variance optimization framework with modern machine learning techniques. We investigate robust portfolio construction methods that address estimation error, regime changes, and practical implementation constraints.

The project is part of the MSCA Digital Finance research program, building on the methodologies developed in IRP16 for financial time series analysis and prediction.

M MVO Methodology

Mean-Variance Optimization (MVO) seeks to construct portfolios that maximize expected return for a given level of risk, or equivalently minimize risk for a target return level.

min w'Sigma*w
s.t. w'mu >= r_target
sum(w) = 1
w >= 0 (optional)

Robust Estimation

Shrinkage estimators and Bayesian approaches to reduce estimation error in covariance matrices.

Factor Models

Multi-factor risk models for improved covariance estimation and risk decomposition.

Regime Detection

Hidden Markov models and changepoint detection for regime-aware portfolio allocation.

Transaction Costs

Incorporating realistic trading costs and turnover constraints into the optimization.
